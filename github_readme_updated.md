# Cyber DNA — Insider Threat Detection via Behavioral Profiling

**Cyber DNA** builds 29-dimensional Digital Behavioral Signatures (DBS) from corporate activity logs — logons, emails, USB events, off-hours patterns — and uses XGBoost to detect insider threats on the CMU CERT r4.2 dataset under a **strict leakage-free chronological protocol** (Train: Weeks 1–52, Test: Weeks 53–72).

The full research paper is in [`final_project_report.md`](./final_project_report.md).  
GitHub: https://github.com/spoorthi2615/Cyber-DNA

## How to Run (3 Steps)

**Prerequisites:** Python 3.9+, Node 18+, CERT r4.2 logs in `data/`

**Step 1 — Run the ML pipeline**
```bash
pip install -r requirements.txt
python cyber_dna_phase11_ablation.py
```
Outputs: `results/phase11_ablation_metrics.csv`, `results/phase11_feature_importance.csv`, `results/phase11_run_summary.json`

**Step 2 — Export results to dashboard JSON**
```bash
python src/export_to_web.py
```
Outputs: `web_app/src/cyber_dna_data.json`

**Step 3 — Run or build the dashboard**
```bash
cd web_app
npm install
npm run dev        # local development at http://localhost:5173
# OR
npm run build      # production bundle in dist/
```

Full deployment instructions (Vercel, Netlify, GitHub Pages) in [`DEPLOYMENT.md`](./DEPLOYMENT.md).

## Dashboard Tabs

The React dashboard is a **read-only presentation layer** — it reads `cyber_dna_data.json` and does not retrain models or recompute features.

| Tab | What it shows |
| :--- | :--- |
| **Overview** | Dataset summary stats and Baseline vs. Expanded model side-by-side comparison |
| **Ablation Study** | Bar chart and full sweep table across all 5 feature configurations |
| **Feature Importance** | XGBoost Information Gain ranking for the top 10 features |
| **Research Results** | Final conclusions, key findings, and simulated Phase 11 execution log |
